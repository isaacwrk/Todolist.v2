> A to-do list built in vue.js, to study vue components and their functions
  

## Project setup

```

npm install

```

  

### Compiles and hot-reloads for development

```

npm run serve

```

  

### Compiles and minifies for production

```

npm run build

```

  

### Run your tests

```

npm run test

```

  

### Lints and fixes files

```

npm run lint

```
